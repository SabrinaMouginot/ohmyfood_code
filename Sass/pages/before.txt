L'utilisation de ::before dans le code CSS crée un pseudo-élément avant le contenu de l'élément HTML spécifié. 
Cela permet d'ajouter du contenu ou des styles supplémentaires à cet élément sans modifier le contenu réel de l'élément HTML lui-même.

Dans le code que j'ai fourni précédemment, ::before est utilisé pour créer un pseudo-élément qui s'ajoute à l'élément <i> (l'icône de cœur) 
et affiche un cœur vide avec un dégradé de couleurs. 
Le contenu du pseudo-élément est défini par la propriété content qui affiche un caractère Unicode correspondant à un cœur vide. 
La propriété display: block; est utilisée pour rendre le pseudo-élément un élément distinct avec sa propre boîte de modèle.

En résumé, ::before est une manière d'ajouter du contenu ou des styles supplémentaires avant l'élément HTML spécifié, 
créant ainsi un effet visuel ou modifiant le contenu de cet élément de manière dynamique en utilisant du CSS. 
C'est couramment utilisé pour ajouter des éléments décoratifs, des icônes, des bordures, etc., 
à des éléments HTML sans avoir à les modifier directement dans le code HTML.
